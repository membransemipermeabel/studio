# **App Name**: PaddyFlow Insights

## Core Features:

- Hero Section with Auto-Scroll: Displays a prominent title and subtitle, with buttons for smooth scrolling to the Dashboard and Background sections.
- Interactive Analytics Dashboard: Embeds a Power BI report within a responsive container, allowing users to visualize and analyze data related to the rice supply chain.
- Loss Root Cause Analysis: Uses AI tool to suggest potential causes and solutions of loss, given identified anomalies. Uses past case data to assess likelihood and confidence of root cause of losses and possible remediation approaches.
- Background Context Section: Provides context on post-harvest losses in the rice supply chain and explains critical data points (harvest, drying, milling).
- Team Members Section: Displays team member information with optional photos, roles, and descriptions, using a responsive grid layout.
- Platform Overview Section: Explains the platform's purpose as an analysis tool for process efficiency and decision-making, highlighting features like mass flow visualization and KPI tracking.
- Sticky Navigation Bar: A navigation bar that remains fixed at the top of the screen, providing smooth scrolling links to each section.

## Style Guidelines:

- Primary background color: Cream / Pale Lemon (#F9FCE3) for a soft, yellowish background.
- Secondary color: Dark Olive (#4B5E36) for headings and lines, providing contrast and readability.
- Decoration color: Soft Sage Green (#DCECCB) for transparent leaf watermark and background circle patterns.
- Highlight color: Forest Green (#2A5835) for small title boxes, adding a darker, cooler green tone.
- Accent color: Fresh Grass Green (#7AB640) as an accent to complement the photos.
- Body and headline font: 'Inter' sans-serif, for a simple and contemporary look. Note: currently only Google Fonts are supported.
- Single-page layout with vertical sections arranged from top to bottom.
- Responsive design ensures the website adapts to different screen sizes, maintaining a consistent and user-friendly experience across devices.
- Smooth scrolling animations provide a fluid transition between sections, enhancing the user experience with subtle and engaging effects (fade-in, slide-in).
